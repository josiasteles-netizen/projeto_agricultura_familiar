# # 🌱 Agricultura Familiar e Sustentável

Projeto educativo desenvolvido com **HTML, CSS e JavaScript** para conscientizar sobre a importância da agricultura familiar sustentável, seus impactos, desafios e soluções para um futuro mais equilibrado.

## 🚀 Funcionalidades

- Página totalmente responsiva
- Menu de navegação intuitivo
- Galeria de imagens
- Quiz interativo com pontuação
- Barra de progresso no quiz
- Certificado ao concluir o desafio
- Carrossel de casos de sucesso
- Sistema de abas informativas
- Pesquisa de conteúdo
- Dark Mode
- Alto contraste e acessibilidade
- Ajuste de tamanho da fonte
- Contadores animados
- Botão "Voltar ao Topo"
- Animações de entrada (Scroll Reveal)

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (ES6)

📁 projeto

## 🎯 Objetivo

Promover a educação ambiental e destacar o papel da agricultura familiar na produção de alimentos, preservação dos recursos naturais e desenvolvimento sustentável.

## 👨‍💻 Autor

Desenvolvido para fins educacionais e de conscientização sobre sustentabilidade no campo.
